A simple, standalone web tool to help students calculate the semester GPA needed to achieve a target CGPA.

This project is built with plain HTML, CSS, and JavaScript, and is deployed live using GitHub Pages.

**View the live tool here:** [https://5uhag.github.io/gpa-calculator/](https://5uhag.github.io/gpa-calculator/)

---

## Features

* **Goal-Seeking:** Instead of just calculating your current GPA, this tool tells you the *exact* GPA you need to hit your target.
* **Instant Calculation:** Runs 100% in your browser using client-side JavaScript. No server required.
* **Clean UI:** A simple, modern, and responsive card-based design.
* **Smart Feedback:** Provides clear error or success messages (e.g., if a goal is mathematically impossible or already met).

## How to Use

1.  Enter your **Current CGPA**.
2.  Enter your **Total Credits Completed** so far.
3.  Enter your **Target CGPA** (the one you want to achieve).
4.  Enter the **Current Semester Credits** you are taking.
5.  Click "Calculate Goal" to see the required semester GPA.
